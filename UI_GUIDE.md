## Dashboard Features

### 1. **Message Input Panel** (Left Side)
- **Text Area**: Enter scam messages to test the system
- **Quick Examples**: Pre-filled example buttons for quick testing:
  - Bank Suspension scam
  - Prize Winner scam
  - UPI Verification scam
  - Phishing Link scam
- **Process Button**: Send messages to the AI system

### 2. **Statistics Panel** (Left Side)
Real-time metrics showing:
- **Messages Processed**: Total number of messages analyzed
- **Scams Detected**: Number of scams identified
- **Intelligence Items**: Total intelligence extracted
- **Avg Engagement**: Average conversation duration

### 3. **Intelligence Summary** (Left Side)
Quick overview of extracted intelligence:
- **Bank Accounts**: Count of bank account numbers found
- **UPI IDs**: Count of UPI payment IDs found
- **Phishing URLs**: Count of suspicious URLs found

### 4. **Conversation View** (Right Side)
- **Real-time Chat**: See messages as they're processed
- **Color-coded Messages**:
  - ðŸ”´ Red bubbles = Scammer messages
  - ðŸ”µ Blue bubbles = AI Agent responses
- **Timestamps**: Each message shows when it was sent
- **Clear Button**: Reset the conversation

### 5. **Agent Response Card** (Right Side)
Shows the latest agent response with:
- Response message text
- Scam detection status
- Agent activation status
- Turn count in conversation

### 6. **Intelligence Details** (Right Side)
Detailed view of all extracted intelligence:
- **Bank Accounts**: List of all account numbers found
- **UPI IDs**: List of all UPI IDs found
- **Phishing URLs**: List of all suspicious URLs found

## How to Use

### Step 1: Open the Dashboard
Navigate to `http://localhost:8000` in your browser

### Step 2: Test with Example Messages
1. Click any "Quick Example" button to fill the input
2. Or type your own scam message
3. Click "Process Message"

### Step 3: View Results
- Watch the conversation appear in real-time
- See statistics update automatically
- Check extracted intelligence in the details panel

### Step 4: Continue Conversation
- Send follow-up messages using the same conversation
- The system maintains context across messages
- Intelligence accumulates as more messages are processed
